# demo/

Hackathon demo assets.

- `ableton/Lichtspiel_Demo_Set.als` — the demo Live Set (clearly-named clips:
  the names drive metadata retrieval). Built in Ableton; not yet committed.
- `clips/` — exported demo clips.
- `captures/` — screen recordings / screenshots (gitignored except `.gitkeep`).
- `scripts/` — capture + setup helpers.

The run-of-show is in `../docs/hackathon_demo_script.md`, including the fallback
ladder (CLI/keyboard → emulator → metadata retrieval → `minimalPulse`).
